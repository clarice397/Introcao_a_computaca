pen drive:E/S. Conecta-se ao sistema operacional via porta USB, é reconhecido como um dispositivo de armazenamento removível. Sua função principal é armazenar e transferir arquivos entre dispositivos

mouse:E. Conecta-se ao sistema por USB ou via conexão sem fio, permitindo que o sistema operacional interprete seus movimentos. Sua função principal é controlar o cursor  com a interface gráfica.

fone de ouvido:S. Conecta-se por entrada P2, USB ou Bluetooth, sendo identificado pelo sistema como dispositivo de saída de áudio. Sua função principal é reproduzir sons de forma privada para o usuário.

microfone:E. Conecta-se por entrada P2, USB ou Bluetooth, sendo reconhecido como dispositivo de entrada de áudio. Sua função principal é captar sons e voz.

webcam :E. Conecta-se geralmente via USB e é detectada pelo SO como dispositivo de vídeo. Sua função principal é capturar imagens e vídeos para chamadas, gravações e transmissões.

1:Se um computador não possuísse dispositivos de saída (como monitor, caixas de som ou impressora), ele continuaria processando dados, mas se tornaria inútil para a interação humana direta.

2: São essenciais pois armazena e transfere informações, de um dispositivo para outro de uma forma eficiente.
